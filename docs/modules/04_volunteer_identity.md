# 🤝 Volunteer & Identity Module

The **Volunteer & Identity Module** manages the community's human capital, focusing on skills mapping and building interpersonal trust.

### Key Features
- **Trust Scores**: Every user starts with a baseline trust score that increases with every successful, rated resource-sharing interaction. 
- **Skill-Based Directory**:
    - Users register with specific skills: First Aid, Electrical, Transport (Heavy Vehicle), Shelter Management, etc.
- **Active Status Tracking**: A real-time volunteer list showing those who are "Active" and "Nearby" vs "Offline."
- **Role Permissions**: Different user tiers (Resident vs Verified Volunteer vs NGO) have different access levels for accepting emergency requests.

### Workflow
1. **Onboarding**: Users complete their profile with area codes and skills.
2. **Participation**: Users respond to nearby "Needs" or "Offers."
3. **Rating**: After completion, the system updates the "Trust Rank" for the participants.
4. **Search**: During a crisis, users can search for specific experts (like "First Aid") in their neighborhood.
